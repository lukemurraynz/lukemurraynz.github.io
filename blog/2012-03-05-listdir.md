This is how you make a list of directories/files on your computer using Command Prompt,

  1. Click Start
  2. Click Programs
  3. Click Accessories
  4. Click Command Prompt
  5. Type in the following command: **dir /a /b /-p /o:gen >filelisting.txt**
  6. It will then create a list.
