World of Warcraft includes a **Mature Language Filter** that replaces profanity in chat with symbols. If you prefer to see unfiltered chat text, you can disable the filter directly from the in-game interface options.

<!-- truncate -->

## Steps to disable the Mature Language Filter

1. Press **Esc** to open the game menu.
2. Click **Interface**.
3. Select the **Social** tab.
4. **Uncheck** the **Mature Language Filter** option.
5. Click **Ok** to save.

The change takes effect immediately — no need to relog or restart the game client. If you ever want to re-enable the filter, follow the same steps and check the option again.
