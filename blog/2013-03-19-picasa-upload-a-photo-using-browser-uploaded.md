Google Picasa's browser-based uploader sometimes has trouble locating photos through the standard file picker. A quick workaround is to paste the full file path directly into the save dialog.

<!-- truncate -->

## Steps to upload using the browser uploader

1. **Right-click** the photo you want to upload in Windows Explorer.
2. Select **Copy Full Path** (or hold **Shift** and right-click to see the option).
3. In the Picasa browser upload dialog, **right-click** inside the file name field.
4. Select **Paste** to insert the full file path.
5. Click **Ok** to begin the upload.

This approach bypasses any issues with the file picker not displaying your photos, and works reliably in most versions of Picasa on Windows.

> **Note:** Google discontinued Picasa in 2016. This article is preserved for historical reference.
